# Markdown Basic Syntax
## Markdown Basic Syntax
- #syntax rendered output will be H1
- ##syntax rendered output will be H2
- ** ** rendered output will be **BOLD**
- ** rendered output will be *Itallic*
- **>** rendered output will be different font style

|Syntax|Output-style|
|------|------------|
|nnnnnn|nnnnnnnnnnnn|
|kkkkk|llllllll|
